INSTALACION

* Para instalar la aplicacion web simplemente descarge los ficheros en una carpeta dentro de su webserver instalado/favorito. => (ejemplo:XAMPP ---> xampp/htdocs/Prueba0).

* Activa Apache en su webserver local y ya podra acceder a la carpeta.

* Una vez alli acceda mediante el localhost a la direccion de la carpeta. 
	=> (ejemplo: http://localhost/Prueba0)
	El proyecto se abrira en el index.php.

* Siga las instrucciones para jugar:
	- En la pantalla inicial inserte el nombre de los jugadores.(Con el boton crear)
	- Una vez tenga tantos como deseen jugar pulsar Empezar.
	- Pulsar el boton nueva palabra.
	- Probar letras hasta adivinar la palabra.


** Si no funcionara o el usuario considera que la instalacion es demasiado tediosa acceda al juego a traves de:
	https://almagra.org/PPB



* La aplicacion web consiste de las tecnologías siguientes:

	- PHP
	- Javascript (con librería JQUERY) y AJAX.
		Debe utilizar un navegador web con javascript y tener la tecnología activada.
